# Logos Alumma — onde usar cada uma

| Arquivo | Formato | Onde usar no site |
|---------|---------|-------------------|
| `logo-full-dark.png` | A + nome, fundo escuro | **Sidebar** (expandida), **TV** (canto), **Login** |
| `logo-full-white.png` | A + nome, fundo branco | **Esqueci-senha**, **cabeçalho do Relatório individual** |
| `logo-a-dark.png` | Só o A, fundo escuro | **Sidebar recolhida**, ícones em áreas escuras |
| `logo-a-white.png` | Só o A, fundo branco | Rodapé do relatório, ícones em fundo claro |
| `logo-alumma-dark.png` | Só nome ALUMMA, fundo escuro | Quando quiser só o nome em fundo escuro |
| `logo-alumma-white.png` | Só nome ALUMMA, fundo branco | Quando quiser só o nome em fundo claro |

O componente `AlummaLogo` em `@/components/AlummaLogo` usa esses arquivos: `variant` = `a` | `alumma` | `full`, `theme` = `dark` | `light`.
